# docs

Carpeta para enunciados, consignas y material provisto por la cátedra.

## Propósito

Usar esta carpeta como fuente oficial del práctico:

- enunciados;
- consignas;
- criterios de evaluación;
- material complementario entregado por docentes;
- scoring o instrucciones oficiales si la cátedra los entrega como documento.

## Reglas de uso

- Mantener los archivos originales sin editar.
- Si se necesita una versión comentada o resumida, crear un archivo nuevo con nombre claro.
- No guardar acá datos crudos, notebooks, scripts, figuras generadas ni presentaciones finales.
- Cuando se cite una consigna, referenciar la ruta del archivo y, si aplica, página o sección.
